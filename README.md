# Credit Card Financial Dashboard 💳

An end-to-end Power BI dashboard analyzing 667K+ credit card transactions to uncover revenue trends, customer behavior, and transaction patterns.

---

## 📌 Key Metrics

| Metric | Value |
|--------|-------|
| Total Revenue | $57M |
| Total Interest Earned | $8M |
| Total Income | $588M |
| Transaction Amount | $46M |
| Transaction Count | 667K |
| Customer Satisfaction Score | 3.19 |

---

## 🔍 Key Insights

- Blue card holders generate $47M out of $57M total revenue
- Businessmen are the top customer segment ($17.7M revenue)
- Bills is the #1 expenditure category ($14M)
- Graduates lead revenue by education level ($23M)
- Q4 has peak transactions (170K) and highest revenue ($14.5M)
- Swipe transactions dominate at $36M vs Chip $17M vs Online $4M
- Married customers generate more revenue ($29M) than Single ($24M)
- Age group 40-50 is the highest revenue segment ($25M)

---

## 🛠️ Tech Stack

- Database: PostgreSQL / MySQL
- Visualization: Microsoft Power BI
- Data Format: CSV to SQL to Power BI

---

## 📂 Dataset Description

| File | Description |
|------|-------------|
| customer.csv | Age, gender, job, income, satisfaction score |
| credit_card.csv | Card category, transactions, fees, utilization |
| cust_add.csv | Week 53 additional customer records |
| cc_add.csv | Week 53 additional card records |

## 📊 Dashboard Highlights

### Customer Report
- Revenue by Age Group, Gender, Marital Status
- Revenue by Education Level, State, Income Group
- Job-wise Revenue and Interest Breakdown

### Transaction Report
- Quarterly Revenue trend with transaction count
- Revenue by Card Category (Blue/Silver/Gold/Platinum)
- Revenue by Expenditure Type and Payment Method
